## 🎓 Architecture

```
ControlledDataAssistant
├── load_csv()
│   └── Loads any CSV file
├── get_csv_metadata()
│   └── Returns column info + sample rows
├── is_address_question()
│   └── Detects address-based queries
├── is_data_question()
│   └── Detects aggregation queries
├── search_csv_smart()
│   ├── Extracts search terms
│   ├── Progressive filtering
│   └── Returns matching rows with logs
├── search_by_address()
│   ├── Handles address queries
│   └── Calls search_csv_smart()
├── answer_company_question()
│   ├── Extracts company name
│   ├── Finds company data
│   └── Returns company info
├── answer_data_question()
│   ├── Handles aggregation
│   └── Returns statistics
└── interact()
    └── Main conversation loop
```
